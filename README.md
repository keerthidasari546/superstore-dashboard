# Superstore Sales Dashboard

## Project Overview
This project presents a Power BI dashboard built using the Superstore dataset to analyze sales, profit, and regional performance.

## Tools Used
- Power BI
- Superstore Dataset

## Dashboard Visualizations
- Sales by Category
- Profit by Category
- Sales by Region
- Sales by Ship Mode
- Sales by Segment
- Profit by State
- Total Sales Card
- Total Profit Card
- Total Quantity Card

## Key Insights
- Technology category has the highest sales.
- The West region contributes significantly to total sales.
- Consumer is the leading customer segment.
- Standard Class is the most commonly used ship mode.
- Profit varies across different states.
